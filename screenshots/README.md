# Screenshots Directory 📸

This directory contains application screenshots for documentation.

## How to Add Screenshots:

### 1. Required Screenshots:
- `main_dashboard.png` - Main application window
- `dijkstra_algorithm.png` - Dijkstra interface with results
- `ford_fulkerson_flow.png` - Ford-Fulkerson with directed graph
- `bellman_ford_algorithm.png` - Bellman-Ford with path visualization
- `welsh_powell_coloring.png` - Graph coloring with colored nodes
- `mst_algorithms.png` - MST with highlighted spanning tree

### 2. Screenshot Guidelines:
- **Resolution:** 1920x1080 or higher
- **Format:** PNG (preferred) or JPG
- **Quality:** High quality, clear visibility
- **Content:** Show both the algorithm interface and graph visualization

### 3. How to Take Screenshots:

#### Main Dashboard:
1. Run `python app/main.py`
2. Configure network (e.g., 15 nodes, 40% density)
3. Take screenshot of the main window

#### Algorithm Screenshots:
1. Open each algorithm from main menu
2. Configure parameters (e.g., V1 → V10)
3. Click "Calculate" to show results
4. Click "View Graph" to show visualization
5. Take screenshot showing both windows

#### Ford-Fulkerson & Bellman-Ford (Directed Graphs):
1. Ensure arrows are visible at edge midpoints
2. Show capacity/weight labels on edges
3. Highlight the flow/path in the graph

### 4. Naming Convention:
- Use lowercase letters
- Use underscores for spaces
- Include algorithm name
- Be descriptive but concise

### 5. Once Screenshots are Added:
- Remove placeholder image links in README.md
- Update with actual screenshot links
- Test that all images display correctly
